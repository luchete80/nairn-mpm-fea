publishplotj.R is a set of functions to create ppltj files, 
includes adding text, arrow and fan plots

read_MPM.R is a set of functions to read output from NairnMPM to R

GaussianRandomFields.Rproj is an RStudio project for running PaperGRF.R to create Gaussian random fields. The fields are 2D rectangles and can be saved as bmp or tab delimiited text files.
